1. Description

The Smart Waste Management System is an IoT-based application designed to monitor garbage bin levels in real-time and optimize waste collection routes. Sensors installed in waste bins send data to a central system, which helps municipal authorities plan efficient garbage collection and reduce environmental pollution.
